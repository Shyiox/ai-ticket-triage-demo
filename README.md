# ai-ticket-triage-demo

Small offline demo that classifies support tickets (urgency + category) and suggests next actions.

## Live Demo
https://shyiox.github.io/ai-ticket-triage-demo/

![AI Ticket Triage Demo Screenshot](assets/screenshot.png)

## Features
- Urgency classification (high / medium / low)
- Category tagging (auth / abuse / billing / general)
- Search + filtering + sorting
- Runs offline with synthetic tickets.json (no API keys)

## How to run locally
Open the Live Demo or run a local static server.
